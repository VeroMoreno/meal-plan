# 🌱 Mealplan!
Javascript-based application that randomly displays a menu of meals (own recipes) from Monday to Friday

### Feature
Developed with ⚡️ React Hooks

### Technologies
* ✨ Node.js
* 💣 React
* 💅🏾 Styled Components

### Funcionality
* Show food list button
* Button sorts meals randomly
* Drag & drop menu
* Nutrient charts

### Dependencies
* chart.js
* react-chartjs-2
* react-beautiful-dnd
* styled-components
